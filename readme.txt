Free Wordpress Slider to display featured posts with amazing transition effects.

== Description ==

TCVN Coin Slider comes with amazing transition effects (permutations and combinations of effects) and is compatible with Wordpress 3.5.1. The widget is built using jQuery. It is very easy to install and configure with lot many features. The module can be used on homepage of a website to showcase the products, services and brand of a company. The module has navigation controls.

Features:

- Image slideshow module has awesome combinations of Transition Effects.
- Auto resize image with module size.
- More Options: Show Title, Description, Link in Title, Slider Effect ...
- Easy to add image, title, description, link ...
- Support all browsers IE7 and above , Firefox , Safari ,  Google Chorme and Opera
- Easy to configure and setup
- Customize User CSS formatting.
- Lightweight, modern and fast-loading design.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE7+, Firefox 2+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Literally unlimited transitions
- Animated Layers
- Easy installation

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'tcvn-coinslider-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'TCVN Coin Slider Widget' to your sidebar
